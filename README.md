# Photon

## How to build

```bash
git clone https://github.com/carlcc/Photon.git --recursive
cd Photon
mkdir build
cd build
cmake ..
cmake --build .
```

## Dependencies:

- SDL2
- libuv
- libzip
- libdx
- glad
- libdshowcapture
- libyuv
- portaudio
- SSBase

You may need to install the dependencies since they are not provided in the source tree:

- SDL2
- libuv
- libzip
